# Responsive Navigation Bar
## Project Overview

This is a small front-end project that demonstrates how to build a responsive navigation bar using HTML, CSS, Flexbox, and Media Queries.
The navigation bar contains a heading and navigation links, and it adapts smoothly to different screen sizes.

The main goal of this project is to understand Flexbox layout, nested flex containers, and responsive design principles.

## Technologies Used

* HTML5
* CSS3
  * Flexbox
  * Media Queries

## Features
* Responsive Design: The layout adjusts automatically for tablets and mobile devices.

* Font Size Adjustment: Heading font size decreases at different breakpoints for better readability.

* Nested Flexbox: Flexbox is used inside another flex container to manage navigation links efficiently.

* Stacking Navigation Links: On smaller screens, navigation links stack vertically for improved usability.
