# CSS Checkerboard Exercise
A simple HTML and CSS exercise to practice layout fundamentals by creating a checkerboard-style grid with alternating red and black squares.
## Key Features
* Checkerboard-style layout
* Alternating red and black squares using CSS
* Fixed-size grid using Flexbox
* No JavaScript â€” pure HTML & CSS
## Tech Stack
* HTML5
* CSS3 (Flexbox & Pseudo Class)
## Key Learnings
* How to structure repetitive UI elements using HTML
* How Flexbox and flex-wrap create grid-like layouts
* How CSS nth-of-type() pseudo class work
* How to apply alternating styles without JavaScript
