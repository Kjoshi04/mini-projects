# Photo Gallery

A simple **Photo Gallery web page** built using **HTML and CSS** that displays a grid of beautiful black & white photographs.  
When you hover over an image, it **zooms in** while the rest of the images **blur**, creating a clean and elegant focus effect.

## Features
- Displays multiple images in a neat grid layout.
- Smooth hover transition: active image enlarges while others blur.
- Uses **Google Fonts (Elms Sans)** for elegant typography.
- Fully responsive design.
